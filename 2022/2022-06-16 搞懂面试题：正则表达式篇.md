## Q1: 使用正则表达式将数字转换为千分位分隔形式
示例：
```
输入：12345678
输出：12,345,678

输入：666
输出：666
```

```javascript
const reg = /(\d)(?=(?:\d{3})+$)/g

console.log('12345678'.replace(reg, '$1,')) // 12,345,678
console.log('666'.replace(reg, '$1,')) // 666
```

涉及的知识点：
- 断言
- 字符串替换
